# Customized Board
## Variables
**custom_params** for stacking user-defined parameters
**_It must be array_**

**PIECE_TYPES**, **PIECE_SYMBOLS**, **PIECE_NAMES**, **ATTACK_MODES** are defined inside **_BaseBoard_** so that user can define custom pieces
